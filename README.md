# SAFEX 1 CLICK MINIG APP

Official repository for SAFEX 1 click mining app.

## Releases

You can download the lates release from (https://github.com/safex/safex_miner/releases)

## Development

#### Backend:

To start project backend

```
$ npm install
$ npm run dev
```

## Build:

Run

```
npm run make-all-installers
```

to make all installer. This will work only on Mac because of Mac.

You can also run

```
npm run make-win-installer
npm run make-mac-installer
npm run make-linux-installer
```
separately.

For linux builds, you will need to have `rpmbuild` available on system (`apt-get install rpm`).
